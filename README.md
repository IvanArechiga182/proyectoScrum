# PROYECTO DE SCRUM DE ADMON. DE PROYECTOS DE SOFTWARE

Bienvenido a AgileMatch, una plataforma interactiva diseñada para
aprender metodologías ágiles de forma divertida. A través de un
juego inspirado en <b>“100 mexicanos dijeron”</b>, pondrás a prueba
tus conocimientos respondiendo preguntas relacionadas con conceptos
ágiles. ¡Mientras más aciertes con las respuestas más comunes o
valoradas, más puntos obtendrás! Ideal para estudiantes, equipos en
formación o cualquier persona interesada en agilidad.
